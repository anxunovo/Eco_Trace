# 用户指南

欢迎来到 **碳循校园**！本指南将帮助您快速了解并上手平台的核心功能，无论是寻找您需要的二手物品，还是分享您闲置的资源，您都可以通过本平台为环保贡献一份力量。

## 1. 用户注册与登录

1. **注册**：
   - 点击首页或导航栏的「登录 / 注册」按钮，进入登录/注册页面。
   - 切换到「注册」标签，填写您的昵称（将作为您的唯一标识）和密码。
   - 提交后，系统将为您创建账户并自动登录，您的默认头像和个人信息也会生成。
2. **登录**：
   - 切换到「登录」标签，输入您注册时的昵称和密码，点击登录即可。
   - 登录后，您将保持登录状态（通过本地浏览器存储会话）。

*注意：为方便演示，系统内置了部分种子用户（例如“李小雨”等），您可以在没有后端支持的本地纯前端模式下通过右上角“身份切换”快速体验不同角色的视角。*

## 2. 如何发布闲置物品

发布闲置物品非常简单，我们通过四步分步表单引导您完成发布：

1. **点击“+ 发布物品”**：进入发布流程。
2. **第一步：上传照片**：
   - 拍摄或选择您的物品照片并上传。
   - **AI 智能识别**：上传图片后，系统将使用 AI 技术（ZhipuAI）自动为您识别物品类别，并提供智能建议，如物品名称、描述、重量等，极大地简化您的填写工作。
3. **第二步：确认类别**：
   - 确认 AI 识别的物品大类（书本资料、衣物鞋包、数码家电等）。
   - 如果类别是“剩余食物”，您还可以选择细分类别（通用、肉类、蔬菜等），这将影响碳减排的估算。
4. **第三步：流转方式与详情**：
   - 设置流转方式（付费转让、免费赠送、物品交换、面议协商）。
   - 确认并完善物品标题和详细描述。
   - （如果是食物，系统会要求您填写最晚领取时间）。
5. **第四步：预估环保贡献与发布**：
   - 系统将根据物品重量和类别实时**估算**本次流转可能带来的碳减排量。
   - 确认信息无误后，点击“发布”。

## 3. 如何浏览、搜索和发现闲置

1. **首页**：在首页，您可以快速了解校园内的累计环保贡献，并通过快捷分类入口浏览最新发布的闲置物品。
2. **发现闲置页面**：
   - 点击底部导航栏或顶部菜单的「发现」。
   - **关键词搜索**：在顶部搜索框输入您想找的物品。
   - **多维筛选**：使用筛选工具，根据分类、交易方式（免费、付费等）、校区和物品状态（全部或仅看未出）进行筛选。
   - **排序**：支持按最新发布排序。
3. **查看详情**：点击感兴趣的物品进入详情页。您可以在此查看物品的所有信息和预估的减碳标签。
   - *（对于食物类物品，详情页会特别展示食品安全风险提示，请注意查看。）*
4. **联系发布者**：
   - 找到喜欢的物品后，点击页面底部的“我想要”或“联系发布者”按钮，系统会显示发布者的联系方式或交接建议。
   - *本平台不提供在线支付、不托管资金、不做担保交易，所有交易请在线下自行协商并注意安全。*

## 4. 完成流转与记录碳减排

当您的物品成功交接给其他同学后：

1. **进入“我的发布”**：
   - 登录后，点击导航栏您的头像，选择「个人中心」，然后点击「我的发布」。
2. **确认已流转**：
   - 在您的活跃发布列表中，找到对应的物品，点击“✅ 确认已流转”。
   - 确认后，物品状态将变更为“COMPLETED”。
   - 此时，该物品预估的碳减排量将**正式计入您的个人环保贡献，并汇总到校园大盘数据中**。

*（注意：只有标记为已流转的物品才会产生碳减排数据记录。删除或自动过期的物品不会计入。）*

## 5. 了解碳减排估算

本平台鼓励大家参与环保。我们使用简化的权威碳排放系数（CLCD/ecoinvent）估算每件物品如果变成二手流转而非废弃/新购，能够节约多少碳排放。

- 所有碳数据均为**估算值**，旨在直观展示环保贡献。
- 碳减排估算受物品种类、重量及特定系数影响。
- 您可以在「减碳看板」中查看全校的总减碳量、分类柱状图，以及最新的流转记录和个人环保贡献排行。

## 6. 移动端使用提示

本平台专为移动设备进行了优化，让您随时随地都能参与校园流转：

- **底部导航栏**：5 区位对称设计（首页 / 发现 / [+发布] / 减碳 / 我的），您可以单手快速切换核心页面。
- **发布便捷按钮**：底部中间凸起的 `+` 按钮，让您在校园任何角落都能快速拍图发布。
- **抽屉式筛选**：在「发现」页，使用底部的筛选抽屉（BottomSheet）可以方便地应用复杂的过滤条件。
- **沉浸式浏览**：物品详情页采用了无顶部边框的全宽大图展示和底部固定悬浮按钮（CTA），确保您拥有最佳的浏览体验。
- **刘海屏适配**：支持 iPhone 等全面屏设备的底部安全区适配。
