# 用户指南

欢迎使用“碳循校园 EcoTrace”。本指南面向普通用户、演示人员和评审，说明如何浏览、发布和理解平台中的减碳数据。

在线体验地址:

```text
https://eco-trace-anxunovo.netlify.app
```

## 1. 平台能做什么

碳循校园用于校园闲置资源流转。你可以在平台上:

- 发布闲置书籍、衣物、电子产品、宿舍用品和剩余食品
- 浏览其他同学发布的闲置物品
- 通过关键词和分类筛选目标物品
- 查看每件物品预计带来的碳减排量
- 在流转完成后记录个人和校园环保贡献

平台只提供信息撮合和环保估算，不提供在线支付、资金托管或交易担保。

## 2. 浏览闲置物品

1. 打开首页。
2. 点击“发现”或进入 `/listings`。
3. 使用搜索框输入关键词，例如“教材”“台灯”“耳机”。
4. 使用筛选项按分类、校区、流转方式和状态过滤。
5. 点击物品卡片进入详情页。

详情页会展示物品图片、标题、描述、发布者信息、流转方式、校区、交接建议、预计减碳量和食品安全提示。

## 3. 发布闲置物品

进入“发布”页面后，按页面引导完成四步:

1. 上传图片。
2. 确认物品分类。
3. 填写标题、描述、成色、校区、交接地点和流转方式。
4. 确认减碳估算并发布。

支持的流转方式:

- 付费转让
- 免费赠送
- 物品交换
- 面议

## 4. AI 辅助功能

如果部署环境配置了 AI Key，上传图片后系统可以辅助生成物品类别、标题、描述、估计重量、价格建议和减碳解释。

如果没有配置后端服务，页面会使用演示模式或本地数据，仍可完成主要流程演示。

## 5. 完成流转

当物品在线下完成交接后，发布者可以在个人中心或我的发布中确认“已流转”。

确认后:

- 物品状态变为已完成
- 预计减碳量计入个人贡献
- 相关数据汇总到校园减碳看板

未完成、删除或过期的物品不会计入正式减碳贡献。

## 6. 理解减碳数据

平台的碳减排量是估算值，核心逻辑是:

```text
减碳量 = 物品重量或数量 x 分类碳排放系数 x 复用折算系数
```

数据适合用于校园环保传播、项目展示、同学参与激励和二手流转活动效果估算，不适合直接作为商业碳交易、审计或认证依据。

## 7. 移动端使用

- 底部导航可快速切换首页、发现、发布、减碳和个人中心
- 发布按钮适合移动端快速拍照上传
- 物品详情页提供大图和底部操作按钮
- 支持 PWA manifest 和 Service Worker 缓存

## 8. 演示建议

1. 打开首页，展示累计数据和项目定位。
2. 进入发现页，展示筛选和物品卡片。
3. 打开一个物品详情页，说明预计减碳量。
4. 进入发布页，演示上传图片和 AI 辅助填写。
5. 进入减碳看板，展示校园贡献和排行榜。
6. 说明平台不做资金交易，只做信息撮合和环保量化。

## 9. 安全提示

- 线下交接建议选择校园公共区域。
- 食品类物品必须确认保质期和保存条件。
- 贵重物品建议当面验货。
- 平台不托管资金，交易风险由双方自行判断。
